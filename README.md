# MapMend Solution

Monorepo with frontend (React + Tailwind) and backend (Node + Express + MongoDB).

## Run locally

### Backend
cd backend
npm install
cp .env.example .env   # edit values
npm run dev            # or node server.js

### Frontend
cd frontend
npm install
cp .env.example .env
npm run dev

## Deploy
- Frontend -> Vercel/Netlify
- Backend -> Render/Heroku/Render
